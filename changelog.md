# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

* * *

## [Unreleased]

## [8.4.0] - 2025-11-10

### Added

- Simplified `robots.txt` file due to web crawlers no longer needing complex rules
- Modules encapsulated into the `lib/modules` folder
- Documentation updates
- Improved Docker setup and enhanced cmds

### Changed

- Added root `lib` folder to match the `modern` app template for consistency

### Fixed

- ColdBox.bx syntax update

## [8.3.0] - 2025-11-10

### Fixed

- Location of ColdBox assets in `server.json` to use `/runtime/lib/coldbox/system/exceptions/` path

## [8.2.0] - 2025-11-01

### Added

- Official ColdBox 8 support and updates
- Enhanced testing capabilities

### Fixed

- ColdBox configuration syntax fixes in `app/config/Coldbox.bx`
- Various template and view improvements

## [8.1.0] - 2025-10-10

### Changed

- Minor updates and refinements
- Version alignment with ColdBox 8 ecosystem

## [8.0.0] - 2025-10-10

### Added

- ColdBox 8 readyness
